Website link -> https://hero-vired-17.web.app

To run the app on local environment, download the project and follow these steps

Step 1 : To install the packages run -> npm install (in root directory)

Step 2 : To start frontend server run -> npm run dev (in frontend directory)

Step 3 : To start backend server run -> npm start (in backend directory)
